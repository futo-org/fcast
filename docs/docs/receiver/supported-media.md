
# Supported Media

The FCast Receiver supports the following media formats. Note that third-party receivers and/or future updates may change the currently supported media formats.

!!! note

    Codec support is dependent on the platform's available hardware/software decoders, such as the Android receiver using the platform's media codecs or the Electron receiver using Chromium supported codecs.

## Streaming Formats
* DASH (Dynamic Adaptive Streaming over HTTP)
* HLS (HTTP Live Streaming)
* WHEP (WebRTC-HTTP Egress Protocol)

## Video Formats
* MP4
* Matroska
* WebM
* 3GP

## Audio Formats
* AAC
* FLAC
* M4A
* MP3
* OGG
* WAV
