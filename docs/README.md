# FCast Documentation Site

This site is built using the [Zensical](https://zensical.org/) static site generator.


# Getting Started

This assumes you have Zensical [installed](https://zensical.org/docs/get-started/) using the [`uv`](https://docs.astral.sh/uv/) package manager.

## Local preview
```
uvx zensical serve
```

## Deployment

#### Setup
```
npm install
```

#### Local preview
```
uvx zensical build
npm run dev
```

#### Publish
```
npm run deploy-site
```
